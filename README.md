# Sales Forecasting Dashboard

A Business Intelligence dashboard built using Streamlit to analyze retail sales data, forecast future sales, detect anomalies, and identify product demand segments.

## Live Demo
🔗 https://sales-bi-dashboard.streamlit.app/

## Features
- Sales overview with interactive KPIs
- 3-month sales forecasting
- Anomaly detection
- Product demand segmentation
- Interactive filters and visualizations

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib

## Project Structure
- `analysis.ipynb` – Data analysis and model development
- `app.py` – Streamlit dashboard
- `requirements.txt` – Required libraries
- `summary.pdf` – Executive business report
- `charts/` – Saved chart images

## Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
